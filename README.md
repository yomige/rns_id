# 帕劳身份证申请教程及用途

[帕劳居民身份证](https://rns.id/?rc_by=joXTZVdN)是由帕劳政府颁发的数字居民身份证，可用于各种身份证明用途。申请帕劳居民身份证之后可以邮寄到国内，在收到实体证件之前，也可以现在电子版来完成KYC操作。

## 帕劳身份证用途

帕劳数字居民身份证的主要用途：

* 身份证明
* 用于完成各种KYC
* 获得帕劳的地址证明
* 用于虚拟货币交易所认证
* 每次入境帕劳可居住180天
* 入住酒店，购买机票登机等
* 帕劳之外的收入不需要向帕劳纳税
* 可在以太坊等公链上拥有合法数字身份

## 帕劳身份证如何申请？

### 方式1: 帕劳身份证官网注册

官网： [https://rns.id/](https://rns.id/zh-CN?rc_by=joXTZVdN)。使用邮箱登录即可。
